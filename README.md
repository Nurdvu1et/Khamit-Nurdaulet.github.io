# Khamit-Nurdaulet.github.ioKhamit-Nurdaulet.github.io

E-Commerce API
This is a simple e-commerce backend built with Node.js, Express.js, and MongoDB.

🚀 Features
✅ User authentication & management
✅ Product CRUD operations
✅ Order management
✅ MongoDB database integration
✅ CORS enabled for API requests

🛠️ Tech Stack
Backend: Node.js, Express.js
Database: MongoDB (Mongoose)
Authentication: JWT (optional)
Environment Variables: dotenv
Deployment: Render
📦 Installation
1️⃣ Clone the repository
bash
git clone https://github.com/Nurdvu1et/Khamit-Nurdaulet.github.io.git
cd Khamit-Nurdaulet.github.io
2️⃣ Install dependencies
bash
yarn install  # or npm install
3️⃣ Create a .env file
plaintext
PORT=5000
MONGO_URI=mongodb+srv://dbUser:dbUserPassword@cluster0.hp3oq.mongodb.net/?retryWrites=true&w=majority&appName=Cluster0
4️⃣ Start the server
bash
npm start  # or yarn start
Server will run on http://localhost:5000/
